## Overview

D-Savor is a restaurant website template featuring a modern design with PHP backend functionality. Includes menu displays, reservation systems, and contact forms.

## Features

- **Homepage** - Hero section with restaurant branding
- **Menu Display** - Organized food menu with categories
- **Reservations** - Table booking functionality
- **Contact Form** - Customer inquiry submission
- **Gallery** - Food and restaurant photos
- **Responsive Design** - Mobile-friendly layout

## Getting Started

### Prerequisites

- PHP 7.4+
- Web server (Apache/Nginx) or PHP built-in server

### Installation

```bash
# Clone the repository
git clone https://github.com/itzkennedydev/D-Savor.git
cd D-Savor

# Start PHP development server
php -S localhost:8000
```

Open [http://localhost:8000](http://localhost:8000) to view the website.
